about the data

The data is taken from the kaggle. It contains 60 records and 7 columns which includes information about tourist places name, region, elevation etc. The insights gained from the analysis will help in understanding about the India tourism. This project describes a new, emerging way to shed light on a dataset and to communicate the findings with a broad audience. I begin this process with understanding of the data, connect the data to the Power BI tool and create visualization. I illustrate the concepts using Indian Tourism data to find the information related to the destinations. In this paper, I describe in detail the process by which i explored the data and built the visualizations that underlie that work.

Inference

Inferences analyzed from the data

  • Radhanagar beach is having the highest rating and Kufri is having the lowest rating.
  • Amarnath Cave is having the highest elevation and Jagganath temple is having lowest elevation.
  • Somnath Temple is the most popular destination.
  • North Region is having maximum railway and airport distance.

process

In order to analyze the data and generate insights out of it, i followed the a process that looked at the data using the following visualizations:

  • Destination wise Rating
  • Destination wise Elevation
  • Top 10 Destinations
  • Region wise Railway and Airport Distance
