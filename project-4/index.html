about the project

concept

This project describes a new, emerging way to shed light on a dataset and to communicate the findings with a broad audience. I begin this process with understanding of the data, connect the data to the Tableau tool, perform joins and create visualization. I illustrate the concepts using hospital data to find inpatient data. In this paper, I describe in detail the process by which i explored the data and built the visualizations that underlie that work.

data

It is a huge dataset of US Healthcare where the information related to hospital inpatients are given. The objective of this project is to analyze the relations between the providers and the beneficiaries.

Inference

Below are the inferences which i analyzed from the dataset

  • Medical Doctors is having the maximum credentials
  • The white race beneficiaries are having highest count whereas other is having lowest
  • The Hypertension is having highest beneficiaries
  • The medicare paid amount for the male providers is maximum
  • The California state is having the maximum beneficiaries
  • Hypertension beneficiaries are highest whereas lung diseases are low

process

In order to analyze the data and generate insights out of it, i followed the a process that looked at the data using the following visualizations:

  • Provider wise Credentials
  • Beneficiaries by race count
  • Depression vs Hypertension
  • Medicare allowed vs payment allowed
  • State wise Beneficiaries
  • Diseases wise Beneficiaries
